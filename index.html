Enter The Location







This weather application is making use of OpenWeather API to fetch in real time weather data.

How To Use?
Enter the name of the city and select the preferred unit to get real-time weather data of the location.

What are the Limits?
The web application is running on the free version of OpenWeather API, which is limited to 1000 API calls a day.

Are there any frameworks used?
Not at all! The website is built using HTML, CSS & Vanilla Javascript. Not to mention VS Code.
